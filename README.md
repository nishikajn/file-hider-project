•	🔐 File Hider System
•	A secure, console-based Java application designed to hide and restore user files by storing them safely inside a MySQL database using LONGBLOB.
This project demonstrates strong understanding of Java backend development, JDBC connectivity, file handling, and database management.
•	________________________________________
•	📌 Project Overview
•	The File Hider System enables authenticated users to securely store files inside a relational database instead of keeping them directly on the system.
It ensures controlled access through user authentication and allows seamless file restoration when required.
•	This project focuses on:
•	Secure file storage
•	Database-driven file management
•	Backend logic implementation
•	Clean modular architecture
•	________________________________________
•	🚀 Key Features
•	🔑 User Registration & Login System
•	📁 Hide files by storing them as LONGBLOB in MySQL
•	♻️ Restore files to their original location
•	🗄️ Database-driven file management
•	🧭 Interactive, menu-driven console interface
•	🔌 Robust JDBC connectivity with proper resource handling
•	📦 Maven-based project structure
•	________________________________________
•	🏗️ Technical Highlights
•	Implemented file storage using PreparedStatement and binary streams
•	Managed large files using MySQL LONGBLOB
•	Applied modular coding principles (separation of concerns)
•	Implemented exception handling for database and file operations
•	Followed clean project structure using Maven
